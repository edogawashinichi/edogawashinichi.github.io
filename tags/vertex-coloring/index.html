Brooks Theorem

Lemma

$\chi(G)\leq\Delta(G)+1$,
that is:
the chromatic number is no more than
the maximum degree plus one.

HINT:
第一感是这个结论不平凡，
$\Delta(G)+1$种颜色local满足每个点的闭邻域，
但如何保证global不会产生矛盾？
给每个闭邻域规定一个量子态$c(N[v])\sim U\binom{[\Delta+1]}{d(v)+1}$，
即每个闭邻域的染色方案等概率取自$\binom{[\Delta+1]}{d(v)+1}$，
让每个闭邻域的量子态以单点独占闭邻域竞争方式坍缩，
这样得到的染色方案是否合法难以决断。
传统的染色算法是
给每个点规定一个量子态$c(v)\sim U\binom{[\Delta+1]}{1}$，
即每个点的染色方案等概率取自$\binom{[\Delta+1]}{1}$，
让每个点以单点独占单点竞争方式坍缩（也就是给点集规定一个访问次序），
这样得到的染色方案一定是合法的。
限定每个量子态坍缩时满足染色条件：相邻点颜色不同，
否则无法完成坍缩得到的染色方案不合法。
$\bigstar$

PROOF:
Let $\sigma=[v_1,v_2,\cdots,v_n]$ be any order of $V(G)$.
Color $V(G)$ with $\Delta(G)+1$ colors in order $\sigma$,
and we’ve done, since the $\Delta(G)+1$ colors suffice
every closed neighborhood
$N[v]=\lbrace v\rbrace\cup N(v)$ for any $v\in V(G)$,
and $V(G)$ is a finite set.
$\square$

Theorem (Brooks, 1941)

If $\chi(G)=\Delta(G)+1$,
then $G$ is either a complete graph,
or an odd cycle.

Or equivalently,
if $G$ is neither a complete graph,
nor an odd cycle,
then $\chi(G)\leq\Delta(G)$.

Definition

Define the set $[k]=\lbrace1,2,\cdots,k\rbrace$,
for any positive integer $k\in\mathbb{Z}^+$.
Define the set $\binom{A}{k}$
$=\brace B\subset A:\vert B\vert=k\rbrace$
for any proper $A$ and $k$.

$G$ is a (simple) graph
(finite, undirected, unweighted,
with no self-cycles and multi-edges).
$V=V(G)$ is the vertex set of $G$.
$E=E(G)$ is the edge set of $G$.
$n=\vert V\vert$ is the order of $G$.
$m=\vert E\vert$ is the size of $G$.

$\chi(G)$ is the chromatic number of $G$.
The chromatic number of $G$ is
the minimum number $k$ such that $G$ is $k$-colorable.
$G$ is $k$-colorable, provided
there exists a function $c:V\longrightarrow[k]$,
such that $c(u)\neq c(v)$ for any $(u,v)\in E$.

$d(v)$ is the degree of vertex $v\in V$,
defined by $d(v)=\vert\lbrace u\in V:(u,v)\in E\rbrace\vert$.
$\Delta(G)$ is the maximum degree of $G$,
defined by $\Delta(G)=\max\limits_{v\in V}d(v)$.
$\delta(G)$ is the minimum degree of $G$,
defined by $\delta(G)=\min\limits_{v\in V}d(v)$.

$G$ is a complete graph,
provided $d(v)=n-1$ for any $v\in V$.

$G$ is a cycle,
provided $d(v)=2$ for any $v\in V$,
and $G$ is connected.
If $n\equiv1(\mod2)$,
then $G$ is an odd cycle,
otherwise $G$ is an even cycle.
